# Insurance-cost analysis 
The goal of this project is to analyze historical insurance data (n=1100) 
consisting of customers lifestyle variables, health-related variables, and 
prior claims history to identify the factors that influence insurance costs and 
inform data-driven decision making.

# Dataset
The dataset for this project called insurance_costs.csv is located in the folder "data"

# Content
The project consists of the following folders:
- `data`: Contains the dataset used for analysis.
- `Reports`: Contains the final report in word and pdf format as well as the Rmarkdown file
that integrates the code and the report.
- `Scripts`: Contains two R script: import_and_clean.R that contains code for the importing 
and data cleaning steps for this project. The second, called script.R that contains code 
for the descriptive statistics, visualizations and regression modelling this project.
`Visualisations`: Contains the visualizations generated for this project.
- `Report`: Contains the final report in word and pdf format as well as the Rmarkdown file
that integrates the code and the report.

# Reproducibility
To run the analysis, install R and R studio on your computer. 
- Clone this repository to your local computer
- Run the script import_and_clean.R to import and clean the data.
- Next, run the script script.R to perform the descriptive statistics, visualizations 
and regression modelling for this project.
- Finally, run the Report.Rmd file to generate the final report in word and pdf format.
